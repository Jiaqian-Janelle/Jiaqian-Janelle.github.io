## Core idea

Builds agentic systems as a coherent end-to-end pipeline: every layer (substrate → alignment
→ reasoning → agent design → coordination → deployment) is essential and interdependent.
Covers RAG, memory, tool use, agent patterns, MCP/A2A protocols, and production frameworks.

## Why I want to read it

A single integrated map of the whole agentic-AI stack — directly on my core interest.
