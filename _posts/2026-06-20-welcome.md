## Writing a new post

Drop a Markdown file into the `_posts` folder named like
`YYYY-MM-DD-some-title.md`. Give it a small front-matter block at the top:

```yaml
---
title: "Your Post Title"
date: 2026-06-21
---
```

Everything below the second `---` is plain Markdown. You get **bold**,
*italics*, [links](https://example.com), lists, and code blocks for free.

> Pull quotes look like this — handy for the line you want people to remember.

That's the whole workflow. Commit, push, and GitHub Pages rebuilds the site.
